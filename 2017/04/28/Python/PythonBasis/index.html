数据类型

  • 整数
  • 浮点数
  • 布尔值
  • 空值
  • 字符串
  • list和tuple
  • dict和set

函数参数

  1. 必选参数
  2. 默认参数
    默认参数必须指向不变参数
  3. 可变参数
    *args
  4. 关键字参数
    **kw

参数组合的定义顺序必须是：必选参数、默认参数、可变参数、关键字参数。任意参数都可以通过类似这样的方式进项定义:func(*args,**kw)
